Contenido que debe ir en cada carpeta:

-archivos: los archivos necesarios en la web (pdf, word, excel, etc).
-controladores: los controladores del patron MVC aplicado.
-css: los archivos de estilo css.
-imagenes: imagenes de la web y galerias.
-incl: librerias y codigos externos.
-js: archivos javascript.
-librerias: librerias creadas propias.
-modelos: los modelos del patron MVC aplicado.
-nbproject: archivos para que funcione en netbeans el proyecto.
-vistas: las vistas del patron MVC aplicado.


para lograr hacer funcionar la prueba deben 
crear la base de datos con el siguiente script:

CREATE SCHEMA IF NOT EXISTS `prueba`;

CREATE  TABLE IF NOT EXISTS `prueba`.`items` (
  `id` INT NOT NULL AUTO_INCREMENT ,
  `nombre` VARCHAR(45) NOT NULL ,
  `color` VARCHAR(45) NOT NULL ,
  PRIMARY KEY (`id`) )
ENGINE = InnoDB;
